Bootstrap 4 Breadcrumbs

add breadcrumbs by adding a single class name.

Bootstrap provides the .breadcrumb class which styles a list into a breadcrumb.

Bootstrap adds separators (by using :before and content) to the list as well as display: inline-blcok to the <li> element -- resutling in that "breadcurmb" effect.

Creating a Breadcrumb

to create a breadcrumb

  • Add class="breadcrumb"to the <ul> element that represents the breadcrumb.
  • Add the class=breadcrumb-itemto the <li> element within the breadcrumb.
  • Also add class="active"to the <li>that represents the current page/secton(the page that the user is currently on) -- which whould almost always be the last crumb in the breadcrumb.

Bootstrap 4 Alerts

Display pre-styled alerts and messages for common user actions.

Bootstrap provides an easy wat to create a stylized alert message box with its alert component.

To create an alert box, use the .alert class along with one of the .alert-* classes to specify the kind of alert.

Info Alert
Warning Alert

Dismissible Alerts

You can create dismissible alerts by adding Bootstrap's .alert-dismissible class, along with a Close button. When the user click the Close button, the alert disappers.

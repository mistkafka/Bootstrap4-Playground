Bootstrap 4 Progress Bars

Provide graphical feedback on the progress of a process. Bootstrap 4 leverages the HTML5 <progress> element for this task.

Bootstrap provides a component for displaying progress bars on a website. Progress bars typically used to provide feedback to the user on the progress of a given process or action. users can visualize instantly how much is complete and how much more there is to go.

Default Progress Bar

To create a default progress bar, use the HTML <progress> element with the .progress class applied.

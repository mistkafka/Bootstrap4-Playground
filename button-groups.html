Vertical Button Group

Replace btn-group with btn-group-vertical to make button group stack vertically.

Button Group Size

Bootstrap enables you to control the size of all buttons within the group (so that you don't need to change each button individually).

To do this, add either of the .btn-group-lg, .btn-group-sm and .btn-group-xs classes to the .button-group class make all buttons within the group large, small, or extral small respectively.

Button Toolbar

You can combine button groups together into a single toolbar if required.

To do this, nest the button groups inside a <div> element with Bootstrap's .btn-toolbar class applied.

Combining Drodowns with Button Groups

You can nest dropdowns within button groups by using a separate button group for the dropdown, then nesting it inside the other one.

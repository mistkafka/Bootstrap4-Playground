Bootstrap 4 Buttons

Bootstrap 4 provides seven styles for buttons -- each serving a different semantic purpose.

Semantic Styles

To style a button, use Bootstrap's .btn class, followed by the desired style. For example, class="btn btn-primary" results in a primary button.

Button Elements

Bootstrap 4's button sytels are designed to be used with the <button> element, but they can also be applied to <input> and <a> elements.

You can also apply the button styles to a <label> element when working with checkboxes and radio buttons (more on that blow).

Button Outlines

Block Level Buttons

A block level button spans the width of its parent. You can create a block level button by adding Bootstrap's .btn-block class.

Disabled Buttons

Bootstrap provides specific styles for the disabeld state(as well as for other states, such as active).

If you need to disable a button that uses an <a> element, you can use Bootstrap's .disabled class.
